About

Esta aplicación de eventos es un proyecto de ejemplo para demostrar el uso de Django como framework web. Permite a los usuarios gestionar eventos, incluyendo la creación, edición y eliminación de eventos. La aplicación está diseñada para ser fácil de usar y proporciona una interfaz intuitiva para la gestión de eventos.
